Instructions for calculating the severity and performing instance segmentation :

Step 1:- Image Path: Please enter the file path in the "Image Path" field.
- Example: /path/to/image.jpg


Step 2:- Annotation Path: If you already have an annotation file in XML or JSON format for the image, please enter its path in the "Annotation Path" field.
- Example XML: /path/to/annotation.xml
- Example JSON: /path/to/annotation.json


Step 3:- Generating Annotation File:
- If you don't have an annotation file, you can generate one using free online tools like IMG LAB or VGG Annotator.
- For XML format: Use a tool like IMG LAB to create the annotations and export them in Pascal VOC XML format. Then provide the path of the generated XML file in the "Annotation Path" field.
- For JSON format: Use a tool like VGG Annotator to create the annotations and export them in JSON format. Then provide the path of the generated JSON file in the "Annotation Path" field.
- Ensure accurate and clear annotations with well-defined bounding boxes and appropriate labels for each annotated object.

Note: If you're using an online tool to generate the annotation file, make sure to follow the tool's instructions for exporting in the desired format.
